## 🛠 Tech Stack & Architecture

- **Frontend**: React, Tailwind CSS v4, Vite, HLS.js
- **Backend**: Node.js, Express (MVC Architecture: modular controllers, routes, and utilities in `src/`), pkg (Executable Bundling)
